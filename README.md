# Angular Course Monorepo

A collection of Angular projects built while following an Angular course. Each folder is a standalone Angular workspace covering different topics and practice exercises.

## Structure

| Project | Description |
|---------|-------------|
| `angular_essentials` | Fundamentals — components, data binding, directives, services, routing. Angular 21. |
| `angular_ESpractice` | Hands-on practice applying the essentials (e.g., investment calculator). Angular 21. |
| `angular_advanced` | Advanced topics — component deep dive, lifecycle hooks, complex patterns. Angular 18. |
| `notes` | Personal notes and quick-reference snippets. |

## Getting Started

Each project is a standard Angular CLI workspace. To work on one:

```bash
cd <project-folder>
npm install
npm start
```

## Notes

- Projects are self-contained and use their own `node_modules`.
- Course notes and reminders are kept in `notes/mightForget.md`.
